A Mini CBT Computer Based Test Engine
MAAUN/24/CBS/0025


Project Description

Mini CBT is a simple web-based Computer Based Test (CBT) application developed using Flask, HTML, and CSS.

The system allows users to answer multiple-choice questions and receive their results automatically.

This project demonstrates basic backend and frontend web development using Python.

this project was developed as part of the COS 202 course assignement.

Features
 
•	Display multiple-choice questions
•	Submit answers through a web interface
•	Automatic score calculation
•	Simple and responsive user interface
•	Organized project structure using templates and static files

Technologies Used

  •	Python
	•	Flask
	•	HTML
	•	CSS
	•	Git & GitHub

  Object-Oriented Programming (OOP)

This project follows basic Object-Oriented Programming concepts in Python.

Examples used in the project include:
	•	Classes to organize quiz questions
	•	Objects to represent individual questions
	•	Encapsulation to manage question data and answers

  Data Structures

The project uses simple data structures to store and manage quiz data.
Exemples:
	•	List – to store multiple questions
	•	Dictionary – to store questions, options, and correct answers

  Standard API Used

The project uses the built-in routing system provided by Flask to create simple web APIs.
Exemples:
	•	/ → Displays the quiz page
	•	/submit → Processes submitted answers
	•	/result → Shows the final score

These routes act as basic web APIs that connect the frontend (HTML) with the backend (Python).
